개발 기간 : 2022년 7월 18일 ~ 2022년 08월 25일

## **[ About ]**

**마포구 청년 일자리 사업단 관리 서비스**

[ 프로젝트 성장배경 ]
> 마포구 청년 일자리 사업단이 시작된지 4년차 입니다.
> 마청단의 대한 정보를 알리고자 홈페이지 제작이 시도되었고, 마청단 홈페이지 관리 서비스는 여기에 공지사항과 직원관리 서비스 및 기타 여러 서버스를 제공하고자 만들어졌습니다.
